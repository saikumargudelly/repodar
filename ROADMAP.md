# Repodar Roadmap

Here's what we're building. Some stuff is done. Some stuff is happening now. Some stuff is still just dreams.

**Latest Update:** March 2026

---

## 📋 What We're Aiming For

Help developers discover emerging AI/ML projects faster, understand what's really trending (not just what's noisy), and make better decisions about what to use.

That's it. That's the mission.

---

## ✅ Already Here (Q1 2026)

- 📊 Real-time leaderboard that actually tracks real stars
- 🏆 TrendScore + SustainabilityScore (two numbers that actually mean something)
- 🔍 Auto-discovery that finds 380+ projects automatically
- 🌐 Shareable widgets for your README
- 💬 Signal explainer (explains *why* a project scored what it did)
- 🔄 Side-by-side comparisons
- 🌍 Language radar (which tech stack is winning?)
- 🤖 Weekly AI-powered analysis

---

## 🚀 Next: Q2 2026 (Discovery & Intelligence)

### 🔨 What We're Actively Working On

**Better Filtering** — "Show me Python projects with 5k+ stars trending this week"
- Filter by: language, category, stars, activity level
- Save filter presets so you don't re-enter them
- GraphQL for the power users

**Smart Recommendations** — "Based on what you like, here are 3 projects you should watch"
- More like this: similar repos to ones you follow
- Predict trends before they explode (ML model for early detection)
- Weekly email digest with personalized picks

**Alerts** — "Tell me when language X trends above 80 score"
- Webhook support (integrate with your tools)
- Slack + Discord notifications
- Custom triggers: threshold-based, category-based, anything

### 🤔 Also Planning

**Community Collections** — Your IMDB for AI/ML repos
- Curate "Top 10 LLMs to watch for Q2"
- Let others vote/react
- Share with your team privately

## 📊 Q3 2026 (Data & Power Users)

**Export Everything** — CSV/JSON download for your analysis tools
- Full historical data for any project
- Batch exports for categories
- API access for builders

**Analytics Dashboard** — See patterns nobody else sees
- Track repo cohorts over time (e.g., "how did agents evolve?")
- Which signals actually predict long-term success?
- Spot emerging trends before the crowd
- Export as fancy PDF reports

**Personalization** — Make the dashboard yours
- Pick which charts matter to you
- Saved searches ("show me these 5 repos daily")
- Dark/light themes
- Custom layouts

**Better APIs** — For people building on top of Repodar
- Higher rate limits for paid users
- Batch fetch 50+ repos in one call
- Webhook reliability (retries, delivery guarantees)
- Full API docs portal

## 🔧 Q4 2026 (Performance & Scale)

**Speed** — Make everything lightning fast

- Compress history data
- Virtual scrolling (load huge lists instantly)

**Reliability** — Know what's working
- Uptime dashboard

- Pipeline health checks
- Error tracking + debugging

**New Platforms** — Beyond GitHub
- GitLab support

- Custom repository sources
- Monorepo tracking (meta-llama has multiple projects)

## 🚀 2027+: The Dream List

If we had unlimited time:

**Ecosystem Health Score** — Beyond trending
- Community health metrics
- Maintainer burnout detection
- Real sustainability, not just activity

**Dependency Tracking** — See the graph
- Which projects depend on which
- Cascade effects when popular repo changes
- Find bottlenecks in the ecosystem

**Developer Insights** — Who's moving things
- Find key contributors across projects
- Skill demand from trending tools
- Job market signals from trending stack

**Research Integration** — Connect papers to code
- Link projects to ArXiv papers
- See which research is getting engineered

**Investment Tracking** — Follow the money
- Link trending repos to funding rounds
- See which categories get funded

**Security** — Scan for risk
- Automated vulnerability detection
- Code quality metrics
- Red flags before they explode
